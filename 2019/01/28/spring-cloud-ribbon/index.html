еҸҜд»ҘзңӢеҲ°пјҢеңЁеҲӣе»әиҜ·жұӮзҡ„ж—¶еҖҷ requestFactory.createRequest(request.getURI(), request. getMethod()); , иҝҷйҮҢзҡ„ request.getURI() дјҡи°ғз”Ё д№ӢеүҚд»Ӣ з»Қзҡ„ ServiceRequestWrapper еҜ№иұЎдёӯйҮҚеҶҷзҡ„ getURI еҮҪж•°гҖӮжӯӨж—¶пјҢе®ғе°ұдјҡдҪҝз”Ё RibbonLoadBalancerClient дёӯе®һзҺ°зҡ„ reconstructURI жқҘз»„з»Үе…·дҪ“иҜ·жұӮзҡ„жңҚеҠЎе®һдҫӢең°еқҖгҖӮ

д»Һ reconstructURI еҮҪж•°дёӯжҲ‘д»¬еҸҜд»ҘзңӢеҲ°пјҢе®ғйҖҡиҝҮ ServiceInstance е®һдҫӢеҜ№иұЎзҡ„ serviceid, д»Һ SpringClientFactory зұ»зҡ„ clientFactory еҜ№ иұЎ дёӯиҺ·еҸ–еҜ№еә” serviceId зҡ„иҙҹиҪҪеқҮиЎЎеҷЁзҡ„дёҠдёӢж–ҮRibbonLoadBalancerContext еҜ№иұЎгҖӮ з„¶еҗҺж №жҚ® ServiceInstance дёӯзҡ„дҝЎжҒҜжқҘжһ„е»әе…·дҪ“жңҚеҠЎе®һдҫӢдҝЎжҒҜзҡ„ Server еҜ№иұЎ пјҢ е№¶дҪҝз”Ё RibbonLoadBalancerContextеҜ№иұЎзҡ„reconsдёғructURIWithServer еҮҪж•°жқҘжһ„е»әжңҚ еҠЎе®һдҫӢзҡ„URIгҖӮ

дёәдәҶеё®еҠ©зҗҶи§ЈпјҢз®ҖеҚ•д»Ӣз»ҚдёҖдёӢдёҠйқўжҸҗеҲ°зҡ„ SpringClientFactory е’Ң RibbonLoadВӯBalancerContext:

  • SpringClientFactory зұ»жҳҜдёҖ дёӘз”ЁжқҘеҲӣе»әе®ўжҲ·з«ҜиҙҹиҪҪеқҮиЎЎеҷЁзҡ„е·ҘеҺӮзұ»пјҢ иҜҘ е·ҘеҺӮ зұ»дјҡдёәжҜҸ дёҖ дёӘдёҚеҗҢеҗҚзҡ„ Ribbon е®ўжҲ·з«Ҝз”ҹжҲҗдёҚеҗҢзҡ„ Spring дёҠдёӢж–ҮгҖӮ
  • RibbonLoadBalancerContext зұ»жҳҜ LoadBalancerContextзҡ„еӯҗзұ»пјҢ иҜҘзұ»з”ЁеҚғеӯҳеӮЁдёҖдәӣиў«иҙҹиҪҪеқҮиЎЎеҷЁдҪҝз”Ёзҡ„дёҠдёӢж–ҮеҶ…е®№е’Ң APIж“ҚдҪңC reconstructВӯ URIWithServer е°ұжҳҜе…¶дёӯд№ӢдёҖ )гҖӮ

д»Һ reconstructURIWithServerзҡ„е®һзҺ°йҖ»иҫ‘дёӯпјҢжҲ‘д»¬еҸҜд»ҘзңӢеҲ°пјҢ е®ғд»ҺServerеҜ№иұЎдёӯиҺ·еҸ–hostе’ҢportдҝЎжҒҜпјҢз„¶еҗҺж №жҚ®д»ҘжңҚеҠЎеҗҚдёәhostзҡ„URIеҜ№иұЎoriginalдёӯиҺ·еҸ–е…¶д»–иҜ·жұӮдҝЎжҒҜпјҢе°ҶдёӨиҖ…еҶ…е®№иҝӣиЎҢжӢјжҺҘж•ҙеҗҲпјҢ еҪўжҲҗ жңҖз»ҲиҰҒи®ҝй—ҙзҡ„жңҚеҠЎе®һдҫӢзҡ„е…·дҪ“ең°еқҖгҖӮ

еҲҶжһҗ еҲ°иҝҷйҮҢпјҢ жҲ‘д»¬е·Із»ҸеҸҜд»ҘеӨ§иҮҙзҗҶжё…Spring CloudRibbonдёӯе®һзҺ°е®ўжҲ·з«ҜиҙҹиҪҪеқҮиЎЎзҡ„еҹәжң¬и„үз»ңпјҢдәҶи§ЈдәҶе®ғжҳҜеҰӮдҪ•йҖҡиҝҮ LoadBalancerinterceptorжӢҰжҲӘеҷЁеҜ№ RestTemplate зҡ„иҜ·жұӮиҝӣиЎҢжӢҰжҲӘпјҢе№¶еҲ©з”ЁSpring Cloudзҡ„иҙҹиҪҪеқҮиЎЎеҷЁLoadBalancerClientе°Ҷд»ҘйҖ»иҫ‘жңҚеҠЎеҗҚдёәhostзҡ„ URIиҪ¬жҚўжҲҗе…·дҪ“зҡ„жңҚеҠЎе®һдҫӢең°еқҖзҡ„иҝҮзЁӢгҖӮ еҗҢж—¶йҖҡиҝҮеҲҶжһҗLoadBalancerClientзҡ„Ribbonе®һзҺ°RibbonLoadBalancerClient, еҸҜд»ҘзҹҘйҒ“еңЁдҪҝ з”ЁRibbonе®һзҺ°иҙҹиҪҪеқҮиЎЎеҷЁзҡ„ж—¶еҖҷпјҢ е®һйҷ…дҪҝз”Ёзҡ„иҝҳжҳҜRibbonдёӯе®ҡд№үзҡ„ILoadBalancerжҺҘеҸЈзҡ„е®һзҺ°пјҢиҮӘеҠЁеҢ–й…ҚзҪ®дјҡйҮҮз”ЁZoneAwareLoadBalancerзҡ„е®һдҫӢжқҘе®һзҺ°е®ўжҲ·з«ҜиҙҹиҪҪеқҮиЎЎгҖӮ

иҙҹеҖәеқҮиЎЎеҷЁ

йҖҡиҝҮд№ӢеүҚзҡ„еҲҶжһҗпјҢ жҲ‘д»¬е·Із»ҸеҜ№ Spring CloudеҰӮдҪ•дҪҝз”Ё RibbonжңүдәҶеҹәжң¬зҡ„дәҶи§ЈгҖӮиҷҪз„¶ Spring Cloudдёӯе®ҡд№үдәҶLoadBalancerClientдҪңдёәиҙҹиҪҪеқҮиЎЎеҷЁзҡ„йҖҡз”ЁжҺҘеҸЈпјҢе№¶дё”й’ҲеҜ№ Ribbonе®һзҺ°дәҶRibbonLoadBalancerClienдёғпјҢдҪҶжҳҜе®ғеңЁе…·дҪ“е®һзҺ°е®ўжҲ·з«ҜиҙҹиҪҪеқҮиЎЎж—¶пјҢжҳҜйҖҡиҝҮRibbonзҡ„ILoadBalancerжҺҘеҸЈе®һзҺ°зҡ„гҖӮ еңЁдёҠдёҖиҠӮиҝӣиЎҢеҲҶжһҗж—¶еҖҷпјҢ жҲ‘д»¬еҜ№иҜҘжҺҘеҸЈзҡ„е®һзҺ°з»“жһ„е·Із»ҸеҒҡдәҶдёҖдәӣз®ҖеҚ•зҡ„д»Ӣз»ҚпјҢ дёӢйқўжҲ‘д»¬ж №жҚ®ILoadBalancerжҺҘеҸЈзҡ„е®һзҺ°зұ»йҖҗдёӘзңӢзңӢе®ғжҳҜеҰӮдҪ•е®һзҺ°е®ўжҲ·з«ҜиҙҹиҪҪеқҮиЎЎзҡ„гҖӮ

AbstractloadBalancer
AbstractLoadBalancerжҳҜILoadBalancerжҺҘеҸЈзҡ„жҠҪиұЎе®һзҺ°гҖӮеңЁиҜҘжҠҪиұЎзұ»дёӯе®ҡд№үдәҶдёҖдёӘе…ідәҺжңҚеҠЎе®һдҫӢзҡ„еҲҶз»„жһҡдёҫзұ» ServerGroup, е®ғеҢ…еҗ«д»ҘдёӢдёүз§ҚдёҚеҗҢзұ»еһӢгҖӮ

  • ALL: жүҖжңүжңҚеҠЎе®һдҫӢгҖӮ
  • STATUS_UP: жӯЈеёёжңҚеҠЎзҡ„е®һдҫӢгҖӮ
  • STATUS_NOT_UP: еҒңжӯўжңҚеҠЎзҡ„е®һдҫӢ

еҸҰеӨ–пјҢ иҝҳе®һзҺ°дәҶдёҖдёӘchooseServer()еҮҪж•°пјҢ иҜҘеҮҪж•°йҖҡиҝҮи°ғз”ЁжҺҘеҸЈдёӯзҡ„ chooseServer (Objectkey)е®һзҺ°пјҢ е…¶дёӯеҸӮж•°keyдёә null, иЎЁзӨәеңЁйҖүжӢ©е…·дҪ“жңҚеҠЎе®һдҫӢж—¶еҝҪз•Ҙkeyзҡ„жқЎд»¶еҲӨж–ӯгҖӮ
иҝҳе®ҡд№үдәҶдёӨдёӘжҠҪиұЎеҮҪж•°гҖӮ

  • getServerList(ServerGroup serverGroup): е®ҡд№үдәҶж №жҚ®еҲҶз»„зұ»еһӢжқҘиҺ·еҸ–дёҚеҗҢзҡ„жңҚеҠЎе®һдҫӢзҡ„еҲ—иЎЁгҖӮ
  • getLoadBalancerStats(): е®ҡд№үдәҶиҺ·еҸ– LoadBalancerStats еҜ№иұЎзҡ„ж–№жі•пјҢ LoadBalancerStats еҜ№иұЎиў«з”ЁжқҘеӯҳеӮЁиҙҹиҪҪеқҮиЎЎеҷЁдёӯеҗ„дёӘ жңҚеҠЎе®һдҫӢеҪ“еүҚзҡ„еұһжҖ§е’Ң з»ҹи®ЎдҝЎжҒҜгҖӮ иҝҷдәӣдҝЎжҒҜйқһеёёжңүз”ЁпјҢ жҲ‘д»¬еҸҜд»ҘеҲ©з”ЁиҝҷдәӣдҝЎжҒҜжқҘи§ӮеҜҹиҙҹиҪҪеқҮиЎЎеҷЁзҡ„иҝҗиЎҢжғ…еҶөпјҢ еҗҢж—¶иҝҷдәӣдҝЎжҒҜд№ҹжҳҜз”ЁжқҘеҲ¶е®ҡиҙҹиҪҪеқҮиЎЎзӯ–з•Ҙзҡ„йҮҚиҰҒдҫқжҚ®гҖӮ

BaseloadBalancer
BaseLoadBalancer зұ»жҳҜе’Ңbbon иҙҹиҪҪеқҮиЎЎеҷЁзҡ„еҹәзЎҖе®һзҺ°зұ»пјҢеңЁиҜҘзұ»дёӯе®ҡд№үдәҶеҫҲеӨҡе…і дәҺиҙҹиҪҪеқҮиЎЎеҷЁзӣёе…ізҡ„еҹәзЎҖеҶ…е®№гҖӮ

  • е®ҡд№үе№¶з»ҙжҠӨдәҶдёӨдёӘеӯҳеӮЁжңҚеҠЎе®һдҫӢServerеҜ№иұЎзҡ„еҲ—иЎЁгҖӮ дёҖдёӘз”ЁеҚғеӯҳеӮЁжүҖжңүжңҚеҠЎе®һдҫӢ зҡ„жё…еҚ•пјҢ дёҖдёӘз”ЁдәҺеӯҳеӮЁжӯЈеёёжңҚеҠЎзҡ„е®һдҫӢжё…еҚ•гҖӮ
  • е®ҡд№үдәҶд№ӢеүҚжҲ‘д»¬жҸҗеҲ°зҡ„з”ЁжқҘеӯҳеӮЁиҙҹиҪҪеқҮиЎЎеҷЁеҗ„жңҚеҠЎе®һдҫӢеұһжҖ§е’Ңз»ҹи®ЎдҝЎжҒҜзҡ„LoadBalancerStats еҜ№иұЎгҖӮ
  • е®ҡд№үдәҶжЈҖжҹҘжңҚеҠЎе®һдҫӢжҳҜеҗҰжӯЈеёёжңҚеҠЎзҡ„IPing еҜ№иұЎпјҢ еңЁ BaseLoadBalancer дёӯй»ҳ и®Өдёә null, йңҖиҰҒеңЁжһ„йҖ ж—¶жіЁе…Ҙе®ғзҡ„е…·дҪ“е®һзҺ°
  • е®ҡд№үдәҶжЈҖжҹҘжңҚеҠЎе®һдҫӢж“ҚдҪңзҡ„жү§иЎҢзӯ–з•ҘеҜ№иұЎIPingStrategy,еңЁBaseLoadBalancer дёӯй»ҳи®ӨдҪҝз”ЁдәҶиҜҘзұ»дёӯе®ҡд№үзҡ„йқҷжҖҒеҶ…йғЁзұ»SerialPingStrategyе®һзҺ°гҖӮ ж №жҚ®жәҗз ҒпјҢжҲ‘д»¬еҸҜд»ҘзңӢеҲ°иҜҘз•ҘйҮҮз”ЁзәҝжҖ§йҒҚеҺҶ ping жңҚеҠЎе®һдҫӢзҡ„ж–№ејҸе®һзҺ°жЈҖжҹҘгҖӮ иҜҘзӯ–з•ҘеңЁеҪ“ IPing зҡ„е®һзҺ°йҖҹеәҰдёҚзҗҶжғіпјҢ жҲ–жҳҜ Server еҲ—иЎЁиҝҮеӨ§ж—¶пјҢ еҸҜиғҪдјҡеҪұе“Қзі»з»ҹжҖ§иғҪпјҢ иҝҷж—¶еҖҷйңҖиҰҒйҖҡиҝҮе®һзҺ° IPingStrategy жҺҘеҸЈе№¶йҮҚеҶҷ pingServers(IPing ping, Server[] servers) еҮҪж•°еҺ»жү©еұ• ping зҡ„жү§иЎҢзӯ–з•ҘгҖӮиЎҘе……пјҡзңӢжіЁйҮҠпјҢе®ҳж–№е»әи®®еҝҪз•ҘиҝҷдёӘжЈҖжөӢеҒҘеә·зӯ–з•ҘпјҢдәӨз”ұeurekaе®һзҺ°пјҢеҲ«зһҺж“ҚеҝғдәҶ
  • е®ҡд№үиҙҹиҪҪеқҮиЎЎзҡ„еӨ„зҗҶи§„еҲҷ IRuleеҜ№иұЎпјҢд»ҺBaseLoadBalancerдёӯchooseServer(Object key) зҡ„е®һзҺ°жәҗз ҒпјҢ жҲ‘д»¬еҸҜд»ҘзҹҘйҒ“пјҢ иҙҹиҪҪеқҮиЎЎеҷЁе®һйҷ…е°ҶжңҚ еҠЎе®һдҫӢйҖүжӢ©д»»еҠЎе§”жүҳз»ҷдәҶ IRule е®һдҫӢдёӯзҡ„ choose еҮҪж•°жқҘе®һзҺ°гҖӮ иҖҢеңЁиҝҷйҮҢпјҢ й»ҳи®Ө еҲқе§ӢеҢ–дәҶ RoundRobжұҹRule дёәе·ҘRule зҡ„е®һзҺ°еҜ№иұЎгҖӮ RoundRobinRule е®һзҺ°дәҶжңҖ еҹәжң¬дё”еёёз”Ёзҡ„зәҝжҖ§иҙҹиҪҪеқҮиЎЎи§„еҲҷ
  • еҗҜеҠЁpingд»»еҠЎ:еңЁBaseLoadBalancerзҡ„й»ҳи®Өжһ„йҖ еҮҪж•°дёӯпјҢдјҡзӣҙжҺҘеҗҜеҠЁдёҖдёӘз”ЁдәҺ е®ҡж—¶жЈҖжҹҘ Server жҳҜеҗҰеҒҘеә·зҡ„д»»еҠЎгҖӮ иҜҘд»»еҠЎй»ҳи®Өзҡ„жү§иЎҢй—ҙйҡ”дёә 10 з§’гҖӮ
  • addServers(List newServers): еҗ‘иҙҹиҪҪеқҮиЎЎеҷЁдёӯеўһеҠ ж–°зҡ„жңҚеҠЎе®һдҫӢеҲ—иЎЁпјҢиҜҘе®һзҺ°е°ҶеҺҹжң¬е·Із»Ҹз»ҙжҠӨзқҖзҡ„жүҖжңүжңҚеҠЎе®һдҫӢжё…еҚ•al1ServerList е’Ңж–°дј е…Ҙзҡ„жңҚеҠЎе®һдҫӢжё…еҚ•newServersйғҪеҠ е…ҘеҲ°newListдёӯпјҢз„¶еҗҺйҖҡиҝҮи°ғз”ЁsetServersListеҮҪж•°еҜ№newListиҝӣиЎҢеӨ„зҗҶпјҢеңЁ BaseLoadBalancer дёӯе®һ зҺ°зҡ„ж—¶еҖҷдјҡдҪҝз”Ёж–°зҡ„еҲ—иЎЁиҰҶзӣ–ж—§зҡ„еҲ—иЎЁгҖӮиҖҢд№ӢеҗҺд»Ӣз»Қзҡ„еҮ дёӘжү©еұ•е®һзҺ°зұ»еҜ№еҚғжңҚеҠЎе®һдҫӢжё…еҚ•жӣҙж–°зҡ„дјҳеҢ–йғҪжҳҜйҖҡиҝҮеҜ№ setServersListеҮҪж•°зҡ„йҮҚеҶҷжқҘе®һзҺ°зҡ„гҖӮ
  • chooseServer(Object key): жҢ‘йҖүдёҖдёӘе…·дҪ“зҡ„жңҚеҠЎе®һдҫӢпјҢеңЁдёҠйқўд»Ӣз»Қзҡ„ж—¶еҖҷпјҢе·Із»ҸеҒҡдәҶиҜҙжҳҺпјҢиҝҷйҮҢдёҚеҶҚиөҳиҝ°гҖӮ
  • markServerDown(Server server): ж Үи®°жҹҗдёӘжңҚеҠЎе®һдҫӢжҡӮеҒңжңҚеҠЎгҖӮ
  • getReachableServers(): иҺ·еҸ–еҸҜз”Ёзҡ„жңҚеҠЎе®һдҫӢеҲ—иЎЁгҖӮ з”ұдәҺ BaseLoadBalancer дёӯеҚ•зӢ¬з»ҙжҠӨдәҶдёҖдёӘжӯЈеёёжңҚеҠЎзҡ„е®һдҫӢжё…еҚ•пјҢ жүҖд»ҘзӣҙжҺҘиҝ”еӣһеҚіеҸҜгҖӮ
